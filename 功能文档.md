# 宠物管理系统核心功能文档

## 1. 跨模块联动跳转功能

### 1.1 会员信息 -> 宠物档案联动跳转
- **功能描述**：在“会员信息管理”模块查阅会员记录时，可以在“宠物档案”列的下拉菜单中直接选择所属的特定宠物。选择后系统将弹窗确认，确认无误即可即时跳转到“宠物数字化健康档案中心”模块。
- **页面展现与定位**：
  - 跳转到宠物模块后，系统会默认**取消搜索过滤**，直接展示店内所有的宠物记录（“显示所有宠物信息”）。
  - 系统会自动根据【会员姓名】和【宠物姓名】进行双重匹配，在所有宠物记录中精准定位到目标宠物的所在行。
  - 成功定位后，表格会自动平滑滚动以居中显示目标宠物记录。
  - 目标宠物整行将被加上柔和的**浅蓝色背景高亮（#ecf5ff）**，并自动设为**默认选中状态**，方便用户快速查看其全量健康信息、后续疫苗记录或管理状态。
- **技术要点**：
  - 自动忽略当前宠物状态可能附带的图标（如🏠、✂️等表情前缀），进行纯名称的比对，防止匹配失效。
  - 前后模块间通过 `sig_requestPetJump(memberName, petName)` 信号槽机制进行事件通信和模块压栈切换。
